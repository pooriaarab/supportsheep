B6 · Voice orb — six states

Single SVG/CSS orb component with state prop. Audio level (0..1) drives subtle scale/glow on top of these base states.

Idle
Pre-connection or paused. Soft grey, slow pulse.
Listening
User is speaking. Blue, gentle wave.
Speaking
AI is speaking. Warm orange, faster pulse synced to audio.
Thinking
Tool call / writer generating. Purple, slow rotate.
Error
Connection lost or audio device fault. Red, fast pulse.
Muted
User muted self. Greyed out, low opacity.

Component contract

interface VoiceOrbProps {
  state: "idle" | "listening" | "speaking" | "thinking" | "error" | "muted";
  audioLevel?: number;   // 0..1 — scales orb and intensifies glow
  size?: "sm" | "md" | "lg" | "xl";   // 48 / 96 / 140 / 200 px
}
